# ShopSync Documentation and Home Page

Welcome to the official documentation and home page repository for **ShopSync**!

**Live site:** https://shopsync.aadish.dev

## About This Repository

This repository contains both the comprehensive documentation and the landing page for ShopSync. It serves as the central hub for:

- **Documentation**: Complete guides covering installation, features, development, and deployment
- **Landing Page**: A modern, responsive website showcasing ShopSync's features and capabilities
- **User Guides**: Step-by-step tutorials for getting started and using advanced features
- **Developer Resources**: Architecture details, API references, and contribution guidelines

## Documentation

Our documentation is built with [MkDocs Material](https://squidfunk.github.io/mkdocs-material/) and includes:

### Getting Started

- Installation guides for Android and web platforms
- Firebase setup instructions
- Quick start tutorials

### Features

- Real-time synchronization
- Smart suggestions and AI integration
- Location-based shopping
- Export and import capabilities
- Home widgets and WearOS support
- Recycle bin for deleted items

### Developer Guide

- Project architecture and structure
- Contributing guidelines
- API documentation
- Deployment instructions

### User Guide

- Authentication and account management
- Creating and managing shopping lists
- Working with items and categories
- Sharing lists with family and friends
- Offline mode functionality
- Settings and customization

**Access the documentation:** Visit [https://shopsync.aadish.dev/docs/](https://shopsync.aadish.dev/docs/) or navigate to the "Docs" tab on the landing page.

## Landing Page

The ShopSync landing page is a React-based single-page application featuring:

- **Home**: Overview of ShopSync with key benefits
- **Features**: Detailed showcase of all features with visuals
- **About**: Information about the project and team
- **Contact**: Ways to get in touch and support
- **Docs**: Direct access to comprehensive documentation

The landing page is built with React and Vite, providing a fast and modern user experience.

# Home Widgets

!!! info "Platform Support"
Home widgets are currently available on **Android** only. iOS widgets coming soon!

ShopSync offers home screen widgets for quick access to your shopping lists.

## Available Widgets

### Quick List Widget

View and add items to your most-used list directly from your home screen.

**Features:**

- Shows top 5 unchecked items
- Tap to check off items
- Add new items quickly
- Auto-updates in real-time

### Multiple Lists Widget

See items from multiple lists at once.

**Features:**

- Displays items from up to 3 lists
- Color-coded by list
- Quick add buttons
- Compact view

## Setting Up Widgets

=== "Android" 1. Long-press on home screen 2. Tap **Widgets** 3. Find **ShopSync** 4. Drag widget to home screen 5. Select list to display 6. Resize as needed

Coming soon!

## Configuration

Tap on a widget to configure:

- **List Selection**: Choose which list to display
- **Number of Items**: 3, 5, or 10 items
- **Theme**: Light, dark, or system
- **Auto-refresh**: How often to update (30s, 1min, 5min)

## Widget Sizes

| Size   | Description | Items Shown |
| ------ | ----------- | ----------- |
| Small  | 2x2         | 3 items     |
| Medium | 4x2         | 5 items     |
| Large  | 4x4         | 10 items    |

## Troubleshooting

### Widget Not Updating

1. Open ShopSync app
2. Go to **Settings** â†’ **Widgets**
3. Tap **Refresh All Widgets**
4. Wait a few seconds

### Widget Shows Error

- Check internet connection
- Ensure you're signed in
- Verify the list hasn't been deleted
- Try removing and re-adding widget

!!! tip "Best Practices" - Use widgets for your most frequently accessed lists - Keep widgets small for faster updates - Enable auto-refresh for real-time updates

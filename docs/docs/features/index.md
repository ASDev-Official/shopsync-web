## Feature Highlights

### Real-time Collaboration

Multiple users can edit the same list simultaneously with changes syncing instantly.

### Offline-First Design

Full functionality without internet. All changes sync automatically when reconnected.

### Smart Item Recognition

ML-powered suggestions for item names, categories, and icons as you type.

### Cross-Platform Support

Works on Android, Web, and WearOS with seamless synchronization.

### Customizable Organization

Categories, list groups, colors, and icons to organize your way.

### Privacy Focused

Your data is encrypted and only accessible by you and invited collaborators.

# Managing Lists

Shopping lists are the heart of ShopSync. This guide covers everything about creating, organizing, and managing your lists.

## Creating a List

### Basic List Creation

1. Tap the **floating action button** (+) on the home screen
2. Enter a list name (e.g., "Weekly Groceries")
3. Optionally customize:
   - **Color** - Choose from 10+ colors
   - **Icon** - Select a custom icon
4. Tap **Create**

## Organizing Lists

### List Groups

Group related lists together:

1. Long-press a list
2. Tap **Add to Group**
3. Choose **Create New Group** or select existing
4. Name your group
5. Tap **Create**

### Sorting Lists

Sort lists by:

- Name (A-Z or Z-A)
- Date Created
- Last Modified
- Item Count

## Sharing Lists

Share lists with family and friends:

1. Open a list
2. Tap **share icon**
3. Choose sharing method
4. Send link to collaborators

See [Sharing Guide](sharing.md) for detailed information.

## Deleting Lists

Deleted lists go to the recycle bin for 30 days:

1. Long-press a list
2. Tap **Delete**

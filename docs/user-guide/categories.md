## Custom Categories

Create categories for your specific needs:

### Adding a Category

1. Open **Settings**
2. Tap **Manage Categories**
3. Tap **Add Category**
4. Enter:
   - Category name
   - Choose icon
   - Select color
5. Tap **Save**

### Editing Categories

1. Go to **Manage Categories**
2. Tap a category
3. Modify name, icon, or color
4. Tap **Save**

### Deleting Categories

1. Go to **Manage Categories**
2. Swipe left on category
3. Tap **Delete**
4. Items in that category become "Uncategorized"

## Assigning Categories

### During Item Creation

1. Add a new item
2. Tap **Category** field
3. Select from the list
4. Complete adding the item

### For Existing Items

1. Open item details
2. Tap **Category**
3. Select new category
4. Tap **Save**

## Category-Based Shopping

Use categories to shop efficiently:

**Store Layout Matching:**

1. Create categories that match your store's layout
2. Sort categories in order you encounter them
3. Shop systematically through the store

**Example for Grocery Store:**

1. Produce (front of store)
2. Bakery (usually near produce)
3. Dairy (back corners)
4. Frozen (center aisles)
5. Pantry (center aisles)
6. Household (back aisles)

## Best Practices

!!! tip "Category Tips" - **Match your main store's layout** - Minimize backtracking - **Use consistent naming** - Easy to remember and find - **Create store-specific categories** - Different stores, different layouts - **Use colors wisely** - Visual differentiation - **Don't over-categorize** - 8-12 categories is usually enough

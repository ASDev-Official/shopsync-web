# Sharing & Collaboration

ShopSync makes it easy to share shopping lists with family, friends, and roommates. Changes sync in real-time across all devices.

## How Sharing Works

When you share a list:

- Everyone sees the same items
- Changes sync instantly
- All collaborators can add, edit, and check items
- The list owner controls sharing permissions

## Sharing a List

### Via Link

1. Open a list
2. Tap the **share icon** (top right)
3. Tap **Share Link**
4. Choose sharing method:
   - Text message
   - Email
   - WhatsApp
   - Copy link
5. Send to collaborators

### Via Email

1. Tap **share icon**
2. Select **Email**
3. Enter recipient's email
4. Optionally add a message
5. Tap **Send**

### Via QR Code

1. Tap **share icon**
2. Select **QR Code**
3. Show QR code to collaborator
4. They scan with their camera

## Accepting Shared Lists

### From Link

1. Click the shared link
2. Opens ShopSync (or prompts to install)
3. Tap **Accept**
4. List appears on your home screen

### From Email

1. Open the email invitation
2. Click **View List**
3. Sign in to ShopSync
4. Tap **Accept**

## Real-Time Collaboration

### Seeing Changes

- Changes appear instantly
- See who added/modified items
- Avatar indicators show active users
- Typing indicators during edits

### Activity Feed

View recent activity:

1. Open a shared list
2. Tap **menu** (⋮) → **Activity**
3. See:
   - Who added items
   - Who checked off items
   - When changes were made

## Managing Collaborators

### View Collaborators

1. Open shared list
2. Tap **collaborators icon**
3. See all people with access

### Remove Collaborator

**List Owner only:**

1. Tap **collaborators icon**
2. Tap collaborator to remove
3. Tap **Remove**
4. Confirm removal

### Leave Shared List

If you're a collaborator (not owner):

1. Open the list
2. Tap **menu** (⋮) → **Leave List**
3. Confirm
4. List removed from your home screen

## Ownership & Permissions

### List Owner

The person who created the list can:

- ✅ Add/edit/delete items
- ✅ Rename the list
- ✅ Change list settings
- ✅ Add collaborators
- ✅ Remove collaborators
- ✅ Delete the list

### Collaborators

People you share with can:

- ✅ Add/edit/delete items
- ✅ Check off items
- ✅ View list details
- ✅ Leave the list
- ❌ Cannot rename list
- ❌ Cannot change settings
- ❌ Cannot manage collaborators
- ❌ Cannot delete list

## Sharing Best Practices

!!! tip "Collaboration Tips" - **Be specific about needs** - Add notes to items - **Check before deleting** - Someone might have added it for a reason - **Communicate changes** - Significant modifications warrant a heads-up - **Respect the owner's organization** - Keep their system intact - **Use comments** - Discuss items before removing

## Privacy & Security

### Who Can See What

- **Private lists** - Only you can see
- **Shared lists** - Only invited collaborators can see
- **Public sharing disabled** - Lists are never publicly visible

### Data Protection

- All data encrypted in transit
- Secure Firebase authentication
- List links expire after 30 days of inactivity

### Revoke Access

To stop sharing immediately:

1. Remove all collaborators
2. Or delete the list
3. Generate new share link if needed

## Multi-Device Sync

Shared lists sync across:

- Android phones
- Android tablets
- WearOS watches
- Web browsers
- Multiple accounts

## Troubleshooting

### Shared List Not Appearing

1. Check internet connection
2. Refresh by pulling down on home screen
3. Sign out and sign back in
4. Verify you're using the correct account

### Changes Not Syncing

1. Check online status (WiFi/cellular)
2. Verify collaborator still has access
3. Refresh the list
4. Check for app updates

### Can't Add Collaborators

1. Verify you're the list owner
2. Check recipient has ShopSync account
3. Ensure valid email address or share link
4. Try alternative sharing method

# Working with Items

Items are the individual entries in your shopping lists. This guide covers everything about adding, editing, and managing items.

## Adding Items

### Quick Add

1. Open a list
2. Tap the **Add Item** button
3. Type the item name
4. Tap **Add** or press Enter

### Detailed Add

1. Tap **Add Item**
2. Fill in details:
   - **Name** (required)
   - **Quantity** (e.g., 2, 1.5 lbs)
   - **Category** (Produce, Dairy, etc.)
   - **Notes** (brand, size, preferences)
   - **Icon** (auto-suggested or custom)
3. Tap **Add**

## Smart Suggestions

As you type, ShopSync provides intelligent suggestions:

- **Item names** - Common grocery items
- **Icons** - Matching visual representations
- **Categories** - Appropriate categorization
- **Quantities** - Typical amounts

Powered by TensorFlow Lite machine learning.

## Editing Items

1. Tap an item to open details
2. Modify any field:
   - Name
   - Quantity
   - Category
   - Notes
   - Icon
3. Tap **Save**

## Checking Off Items

Mark items as purchased:

- **Single tap** - Check/uncheck the checkbox
- **Quick check** - Checked items move to bottom
- **Real-time sync** - Everyone sees updates instantly

## Deleting Items

### Single Item

Swipe left on the item and tap **Delete**.

### Multiple Items

1. Long-press an item
2. Tap additional items to select
3. Tap **delete icon**
4. Confirm deletion

Deleted items go to the recycle bin.

## Item Details

Each item stores:

- Name
- Quantity
- Category
- Notes
- Icon
- Checked status
- Created date
- Modified date
- Created by (user)
- Modified by (user)

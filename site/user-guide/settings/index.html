Settings & Preferences¶

Customize ShopSync to match your preferences and workflow.

Accessing Settings¶

  • Mobile: Tap profile icon → Settings
  • Web: Click profile picture → Settings

Account Settings¶

Profile¶

  • Display Name - Change how your name appears
  • Email - View registered email (cannot change)
  • Profile Picture - Syncs from Google account
  • Account Created - When you joined ShopSync

Security¶

  • Change Password - Update your password (email/password accounts)
  • Two-Factor Authentication - Managed through Google account
  • Sign Out - Log out of current session
  • Delete Account - Permanently remove account and data

Appearance¶

Theme¶

Choose your preferred color scheme:

Matches your device's theme setting

Bright, clean interface for daytime use

Easier on eyes, saves battery on OLED screens

Display Options¶

  • Density - Compact, Normal, or Comfortable
  • Font Size - Small, Medium, Large, Extra Large
  • Card Style - Flat or Elevated
  • Animations - Enable/disable transitions

Lists & Items¶

Default Settings¶

  • Default View - Grid or List layout
  • Sort Order - How lists are sorted on home screen
  • Item Sorting - Default item sort within lists
  • Auto-Check Behavior - Where checked items go
  • New List Color - Default color for new lists

Smart Features¶

  • Smart Suggestions - AI-powered item suggestions
  • Auto-Categorize - Automatically assign categories
  • Icon Suggestions - Suggest icons for items
  • Quantity Presets - Common quantity shortcuts

List Options¶

  • Compact Mode - Show more items on screen
  • Show Checked Items - Keep or hide completed items
  • Item Animations - Smooth transitions
  • Swipe Actions - Customize swipe gestures

Offline & Sync¶

Sync Settings¶

  • Auto-Sync - Sync automatically when online
  • Sync Frequency - How often to check for updates
  • WiFi Only - Don't use cellular data for sync
  • Background Sync - Sync when app is closed
  • Battery Saver - Reduce sync frequency on low battery

Storage¶

  • Cache Size - Maximum local storage to use
  • Auto-Clear - Automatically clear old data
  • Download Images - Cache item icons offline
  • Compress Data - Use less storage space

View Storage Usage¶

See what's using space:

  • Total cache size
  • Number of cached lists
  • Number of cached images
  • Pending sync items

Clear Cache:

  • Cached images
  • Old deleted items
  • Unused list data

Language & Region¶

Language¶

Select your preferred language:

  • English
  • Spanish (Español)
  • French (Français)
  • German (Deutsch)
  • And more via Crowdin

Region Settings¶

  • Date Format - MM/DD/YYYY or DD/MM/YYYY
  • Time Format - 12-hour or 24-hour
  • First Day of Week - Sunday or Monday
  • Measurement Units - Imperial or Metric

Privacy¶

Data Collection¶

  • Analytics - Help improve ShopSync (anonymous)
  • Crash Reports - Automatic error reporting
  • Usage Statistics - Feature usage tracking

Data Management¶

  • Export Data - Download all your data
  • Delete History - Clear activity logs
  • Manage Permissions - Review app permissions

Integrations¶

Home Widget (Android)¶

Configure home screen widgets:

  • Select which list to display
  • Choose widget size
  • Update frequency
  • Theme (follows app or custom)

WearOS¶

Sync settings for smartwatch:

  • Enable WearOS companion
  • Sync frequency
  • Display options
  • Completion notifications (not available)

WearOS Capabilities

WearOS app allows viewing lists and completing/uncompleting items only. All other operations require the phone or web app. Notifications are not supported.

Voice Assistants¶

(Coming soon)

  • Google Assistant integration
  • Alexa integration
  • Siri Shortcuts

Advanced¶

Developer Options¶

  • Debug Mode - Show technical information
  • Verbose Logging - Detailed logs for troubleshooting
  • Sync Diagnostics - View sync status details
  • Feature Flags - Enable experimental features

Backup & Restore¶

  • Auto Backup - Automatic cloud backup
  • Backup Frequency - Daily, Weekly, or Manual
  • Restore from Backup - Recover from previous state
  • Export Backup - Download backup file

Reset Options¶

  • Reset Settings - Restore default settings
  • Clear Cache - Remove all cached data
  • Reset Tutorial - Show welcome screens again
  • Factory Reset - Remove all local data

Destructive Actions

Clearing cache and factory reset remove local data. Ensure you're online and synced before using these options.

Accessibility¶

Visual¶

  • High Contrast - Increased contrast for better visibility
  • Large Text - Bigger font sizes
  • Bold Text - Heavier font weight
  • Reduce Motion - Minimize animations

Interaction¶

  • Touch Delay - Longer press duration
  • Haptic Feedback - Vibration on interactions
  • Sound Effects - Audio feedback
  • Screen Reader - TalkBack/VoiceOver optimization

Best Practices¶

!!! tip "Recommended Settings" - Enable Auto-Sync - Keep data current - Use System Theme - Matches your device - Enable Smart Suggestions - Faster item entry - Regular Backups - Protect your data

Troubleshooting¶

Settings Not Saving¶

  1. Check internet connection
  2. Ensure you're signed in
  3. Try closing and reopening settings
  4. Clear app cache
  5. Reinstall app if persistent

Sync Issues¶

See Offline Mode troubleshooting section.

Missing Features¶

Some features may be:

  • Platform-specific (Android only, Web only)
  • Account-type specific (Google vs. Email)
  • Region-locked
  • Experimental (requires opt-in)
